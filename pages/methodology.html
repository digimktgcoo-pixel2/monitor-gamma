Methodology

How we collect and display data

Data Source

All readings come from the EPA RadNet Central Data Exchange (CDX) REST API. This provides near-real-time CSV data from EPA's 140+ fixed air monitoring stations across all 50 states.

API Endpoint

We query the following endpoint for each station:

https://radnet.epa.gov/cdx-radnet-rest/api/rest/csv/{year}/fixed/{STATE}/{CITY}

Status Classification

StatusThresholdMeaning
NORMAL< 100 nSv/hWithin typical US background range
ELEVATED100–149 nSv/hAbove average but within known variation
ALERT≥ 150 nSv/hSignificantly above normal — verify via EPA
NO DATA—Station offline or data unavailable
These thresholds are for display purposes only. They are not official health guidance. Normal background radiation varies by location. See EPA and NRC for authoritative thresholds.

Fallback Data

When a live EPA fetch fails (network timeout, station offline, CORS restriction), the platform displays an estimated reading based on historical baseline values for that region. Estimated readings are clearly labeled.

Refresh Rate

The dashboard automatically refreshes data every 60 minutes. EPA typically updates station readings hourly.

Units

nSv/h — nanosieverts per hour. The standard unit for ambient gamma radiation dose rate. Typical US background is 60–120 nSv/h depending on elevation and geology.

CPM — counts per minute. Raw gamma particle detection count from the monitor sensor.
