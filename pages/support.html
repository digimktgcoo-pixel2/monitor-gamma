How to Use the Dashboard

The main dashboard displays a map of the United States with colored dots representing EPA radiation monitoring stations. Tap or click any dot to see the current radiation reading, status, and plain-English explanation for that location.

Status Level Meanings

ColorStatusRangeMeaning
●NORMAL<100 nSv/hWithin typical US background radiation range.
●ELEVATED100–149 nSv/hAbove average but not unusual for high-altitude areas.
●ALERT≥150 nSv/hSignificantly above normal. Verify via epa.gov/radnet.
●NO DATA—Station is offline or data is unavailable.
These are display indicators only. They are not official emergency alerts.

Data Timestamps and Refresh

EPA updates station readings approximately every hour. This dashboard automatically refreshes data every 60 minutes. The timestamp shown in each station panel reflects when EPA last posted data for that location.

Stale or Missing Data

If a station shows "No Data" or "Estimated," the live EPA fetch for that location failed. This can happen due to station maintenance, communication outages, or temporary EPA server issues. Try refreshing the page after a few minutes. The dot will remain gray until live data is available.

Browser Compatibility

This app works in all modern browsers: Chrome, Firefox, Safari, Edge. Internet Explorer is not supported. For best mobile experience, use the latest version of Chrome (Android) or Safari (iPhone).

Install as a Mobile App (Android)

  1. Open the site in Chrome on your Android device.
  2. Tap the three-dot menu in the top right corner.
  3. Tap "Add to Home screen."
  4. Tap "Add" to confirm.
  5. The app will appear on your home screen and launch in fullscreen.

Install as a Mobile App (iPhone)

  1. Open the site in Safari on your iPhone.
  2. Tap the Share button (box with arrow) at the bottom of the screen.
  3. Scroll down and tap "Add to Home Screen."
  4. Tap "Add" in the top right.
  5. The app appears on your home screen.

Desktop Access

Open any modern browser and navigate to this site's URL. The map and panels are fully functional on desktop. Use the search bar to find specific cities.

Troubleshooting

Map doesn't load

Check your internet connection. The map tiles require an active connection on first load. If connected, try a hard refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac).

All stations show gray dots

Data is still loading. Wait 15–30 seconds for the batch fetch to complete. The progress bar at the bottom shows loading status.

App shows old data

Hard refresh the page or clear browser cache. PWA cache can sometimes serve older versions of the app shell.
